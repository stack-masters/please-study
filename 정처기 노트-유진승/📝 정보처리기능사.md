# 정보처리기능사 합격 필기 / 실기 - 합격 전략 - 유진승

    정보처리기능사 소개
```
정보처리기능사는 과학기술정보통신부에서 주관하고 한국산업인력공단에서 시행하는 정보통신기술 분야의 기능사 자격증으로 정보처리에 있어서 가장 기초적인 것을 테스트하는 자격증시험이다.
```
<br />

    난이도

```
난이도는 필기는 열심히 노력하면 합격할 수 있는다고 본다. 그러나 실기가 고비인것 같다. 합격률을 보면

2022년 1회실기 1.7%확률, 2023년 2회실기 1.6%, 2023년 3회실기 27.6%로 상당히 낮은 합격률을 보인다.

​

필자는 2023년 3회실기에서 합격했었다. 3회실기 시험에대해서는 열심히 공부했으면 합격했을만한 시험이라 나는 생각한다.
```

    필기 공부법

```
정보처리기능사 필기시험준비는 시나공에 정보처리기능사 책을 하나사서 공부했었다.

반복적으로 몇번을 봐서 통암기를 하고 시험장에 가서 시험을 치러 1번에 80점이상을 취득했다.

또 CBT사이트를 이용해서 문제를 여러번 풀었었다.
```
<br />

    실기 공부법
```
정보처리기능사 실기 공부도 마찬가지로 통암기를 하였다. 실기는 필기와 같이 컴퓨터로 시험을 보는게 아니라 직접 1시간 30분동안 적어야하는 시험이기에 암기를 할 수 밖에 없었다.

마찬가지로 시나공에 정보처리기능사 실기책을 사서 통암기를 했었다.

시나공 정보처리기능사 책에 맨뒤에 있는 기출문제를 여러분 풀기는 강력히 추천한다.


마지막으로 실기때 공부했던 노트를 올리려한다.
```
